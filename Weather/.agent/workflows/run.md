description: How to run the Weather Project
---

To run this project, follow these simple steps:

### Option 1: Direct Open (Easiest)
1. Navigate to the project folder: `c:\Users\aksha\OneDrive\Desktop\whether\Weather`
2. Right-click on `login.html`.
3. Select **Open with** and choose your preferred web browser (Chrome, Edge, or Firefox).

### Option 2: Live Server (Recommended for Development)
If you are using Visual Studio Code:
1. Open the project folder in VS Code.
2. Install the **Live Server** extension (by Ritwick Dey).
3. Right-click on `login.html` and select **Open with Live Server**.
4. This will launch the site and automatically refresh when you save code changes.

### Option 3: Terminal (Using Node.js)
If you have Node.js installed:
1. Open your terminal in the project directory.
2. Run the following command:
```powershell
npx serve
```
3. Open the provided local URL (usually `http://localhost:3000`) in your browser.

> [!NOTE]
> The project is configured to start with the **Login Page**. Once you login (use any valid email and a 6-character password), you will be redirected to the main weather dashboard.
