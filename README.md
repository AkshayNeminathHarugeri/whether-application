# Weather Application

A premium weather dashboard that provides real-time weather information, including temperature, humidity, wind speed, and various other metrics.

## Features

- **Taluka-wise Coverage**: Supports weather data for talukas in Karnataka, Maharashtra, and Tamil Nadu.
- **Detailed Metrics**: Provides information on precipitation, pressure, visibility, and sunrise/sunset times.
- **Micro-animations**: Enhanced user experience with smooth transitions and interactive elements.
- **Modern UI**: Sleek, glassmorphic design system.

## Project Structure

- `index.html`: Main dashboard entry point.
- `login.html`: Secure login page.
- `script.js`: Core application logic and API integrations.
- `style.css`: Premium design system styles.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/AkshayNeminathHarugeri/whether-application.git
   ```
2. Open `login.html` or `index.html` in your web browser.

## Technologies Used

- HTML5
- Vanilla CSS3
- JavaScript (ES6+)
